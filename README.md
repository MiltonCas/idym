# Sitio web IDYM — nueva propuesta

Página completamente reestructurada con una apariencia seria, técnica y corporativa.

## Archivos

- `index.html`
- `css/styles.css`
- `js/main.js`
- `assets/`

## Incluye

- Hero profesional con fotografía real.
- Servicios reorganizados.
- Galería de proyectos.
- Espacio para añadir nuevas evidencias.
- Sección institucional.
- Proceso de trabajo.
- Formulario de cotización.
- Diseño adaptable para celular.
- Menú móvil y botón de WhatsApp.

## Antes de publicar

1. Sustituye el enlace del aviso de privacidad.
2. Confirma el correo utilizado en FormSubmit.
3. Agrega enlaces reales de redes sociales si los necesitas.
4. Convierte las fotografías a WebP para reducir el peso.
